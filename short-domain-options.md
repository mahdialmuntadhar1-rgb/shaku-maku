## 🔧 Setup Process for Firebase Hosting

### Step 1: Create Firebase Project (3 minutes)
1. **Go to**: https://console.firebase.google.com/
2. **Click**: "Add project"
3. **Project name**: `shakumaku`
4. **Continue** through setup
5. **Create project**

### Step 2: Enable Hosting (2 minutes)
1. **Go to**: Build → Hosting
2. **Click**: "Get started"
3. **Select**: "I want to host a web app"
4. **Continue** through setup

### Step 3: Deploy Your App (5 minutes)
1. **Install Firebase CLI**: `npm install -g firebase-tools`
2. **Initialize**: `firebase init hosting`
3. **Deploy**: `firebase deploy`
4. **Visit**: `https://shakumaku.web.app`
