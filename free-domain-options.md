## 🔧 Setup Process for shakumaku.tk

### Step 1: Register Domain
1. Go to: https://www.freenom.com/
2. Search for: `shakumaku.tk`
3. Click "Get it now"
4. Create free account
5. Complete registration (2 minutes)

### Step 2: Configure DNS
1. **Login** to Freenom dashboard
2. **Go to DNS Management**
3. **Add CNAME record**:
   ```
   Type: CNAME
   Name: @
   Value: billboard3dnakedeye-mor.mahdialmuntadhar1.workers.dev
   TTL: 3600
   ```

4. **Add WWW record**:
   ```
   Type: CNAME
   Name: www
   Value: billboard3dnakedeye-mor.mahdialmuntadhar1.workers.dev
   TTL: 3600
   ```

### Step 3: Update Application
1. **Update manifest.json**:
   ```json
   "start_url": "https://shakumaku.tk/",
   "scope": "https://shakumaku.tk/"
   ```

2. **Update App.tsx**:
   ```javascript
   url: 'https://shakumaku.tk/',
   ```

3. **Deploy** with new domain
